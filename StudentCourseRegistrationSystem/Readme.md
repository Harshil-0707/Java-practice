# Student Course Registration System (Java)

This is a simple Java console project where students can register into courses, pay fees and view their details.  
This project is made to practice Core Java + OOP concepts.

---

## Features

- Add new Student
- Add new Course
- Register student into one or multiple courses
- Pay course fees (partial or full)
- View student details with remaining fees
- Exit the application

---

## Concepts you will practice

- Classes & Objects
- Constructors
- Encapsulation (private fields + getters/setters)
- ArrayList
- Inheritance (optional)
- Packages
- Method calling between classes

---

## How to use

1. Clone / download this project
2. Open in VS Code / IntelliJ / Eclipse
3. Compile the `.java` files
4. Run `App.java`
5. Use the menu that appears in the console to perform actions

---

## Suggested classes

- `Student` (stores student info + registered courses)
- `Course` (stores course name + fee)
- `RegistrationSystem` (handles all logic of adding student, course, registration etc.)
- `App` (contains `main` and menu)

---
